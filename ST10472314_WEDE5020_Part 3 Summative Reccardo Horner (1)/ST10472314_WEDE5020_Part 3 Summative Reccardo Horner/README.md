# Konoha Ninja Store Website

## Description
Konoha Ninja Store is a fan-based e-commerce website offering authentic ninja gear and Naruto-themed merchandise. The website is fully responsive, visually engaging, and designed to meet both Part 2 and Part 3 requirements. Users can explore products, learn about the store, submit inquiries, and contact the store easily.

This project includes **dynamic theme switching**, **interactive forms with validation**, and improved SEO best practices.

---

## Features

### General Features
- External CSS styling (`gstyle.css`) for all pages
- Consistent typography, colors, and layout
- Fully responsive using Flexbox and media queries
- Navigation bar with hover effects
- Interactive cards for images and featured sections
- Product gallery layout
- Gradient header backgrounds for visual appeal
- All images have descriptive alt text for accessibility and SEO

### Part 3 – Dynamic Enhancements
- **Theme Switching:** Website alternates automatically every 10 seconds between original orange theme and Akatsuki black/red theme using CSS variables and JavaScript. Smooth transitions applied across header, background, cards, buttons, and text.
- **Form Functionality & Validation:** Inquiry form validates user inputs before submission and displays a confirmation alert. Contact page displays contact info; can be extended for email submission.
- **JavaScript Enhancements:** Dynamic theme switching and form validation with alert feedback.
- **SEO Best Practices Implemented:** Proper title tags and heading structure, alt text for all images, clean descriptive URLs, internal linking, and mobile-friendly responsive design.

---

## Pages
- `index.html` – Home page with featured content and images
- `about.html` – About Us page with team and inspiration images (`NinjaGear.webp`, `FiveKage.jpg`, `NarutoUzumaki.webp`)
- `products.html` – Product listing with a responsive gallery
- `inquiry.html` – Interactive inquiry form with client-side validation
- `contact.html` – Contact page with address, email, and phone
