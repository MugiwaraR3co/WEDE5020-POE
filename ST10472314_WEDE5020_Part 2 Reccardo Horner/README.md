# Konoha Ninja Store Website

## Description
Konoha Ninja Store is a fan-based e-commerce website offering authentic ninja gear and Naruto-themed merchandise. The website is fully responsive, visually engaging, and designed following Part 2 CSS and responsive design requirements. Users can explore products, learn about the store, submit inquiries, and contact the store easily.

## Features
- External CSS styling (`gstyle.css`) for all pages
- Consistent typography and color scheme
- Responsive layout using Flexbox and media queries
- Navigation bar with hover effects
- Interactive cards for images and featured sections
- Responsive product gallery
- Styled forms with hover effects
- Gradient header backgrounds for visual appeal
- Fully tested for desktop, tablet, and mobile devices

## Pages
- `index.html` – Home page
- `about.html` – About Us
- `products.html` – Products listing
- `inquiry.html` – Inquiry form
- `contact.html` – Contact information

## Responsive Testing
Screenshots have been taken for different devices:  
- **Desktop:** Full layout with multi-column gallery and centered cards  
- **Tablet:** Navigation stacks vertically, images and text scale appropriately  
- **Mobile:** Single-column layout, full-width images, buttons and forms scale to screen
